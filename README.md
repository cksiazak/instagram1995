# Nextragram95

Nextagram95 is an instagram clone at it's core, but styled to appear as if it were a Windows95 application. I built this to test Nextjs and decided to have fun with it.

# Stack

Frontend:

- Nextjs
- Redux
  - Redux-Persist
  - Redux-Thunk
- Styled Components

Backend:

- Nodejs/Hapi
  - bcryptjs
  - JSON Web Token
- Sequelize
- PostgreSQL

Currently a work in progress
